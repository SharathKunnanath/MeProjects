\section{Recursive Solution}
\begin{flushleft}
    
Time Complexity:
\[T(n) = n \, \textrm{operations} \times  O(1) \]
\[T(n) = O(n) \]

Space Complexity:
\[S(n) = n \, \textrm{dictionary entries} \, \times \, \textrm{size of dictionary element} \]
\[S(n) = O(n) \]
